# 깃허브 업로드 파일 목록 (GitHub Upload List)

이 문서는 깃허브(GitHub) 저장소에 실제로 올라가는 **핵심 코드와 파일**들을 정리한 목록입니다.
민감한 정보나 불필요한 대용량 파일은 제외되고, 프로젝트 실행에 꼭 필요한 파일들만 선별됩니다.

## 1. 📂 핵심 소스 코드 (`src/`)
프로젝트의 주요 로직이 담긴 파이썬 코드들입니다.
*   **위치**: `src\` 폴더 전체 (d:\빅프로젝트\src\)
*   **주요 파일**:
    *   `src/process_all_sr_interp.py`: **[메인 실행 코드]** 위성영상 초해상화 및 보간 처리를 담당합니다.
    *   `src/utils/`: **[보조 함수]** 데이터 처리나 편의 기능을 모아둔 폴더입니다.
    *   (그 외 `src` 폴더 내의 모든 `.py` 파일들이 포함됩니다)

## 2. ⚙️ 설정 및 환경
프로젝트 실행 환경과 모델 설정을 정의하는 파일들입니다.
*   **위치**: 루트 폴더 (`d:\빅프로젝트\`) 및 `configs\` 폴더
*   **주요 파일**:
    *   `configs/`: **[모델 설정]** AI 모델 파라미터나 경로 설정 파일들 (YAML, JSON 등).
    *   `requirements.txt`: **[라이브러리 목록]** 프로젝트 실행에 필요한 설치 패키지들을 명시한 파일입니다.

## 3. 📄 설명 문서 (`docs/` & Root)
프로젝트 사용법과 가이드 문서입니다.
*   **위치**: 루트 폴더 (`d:\빅프로젝트\`) 및 `docs\` 폴더
*   **주요 파일**:
    *   `README.md`: **[프로젝트 대문]** 저장소에 들어왔을 때 가장 먼저 보이는 프로젝트 소개글입니다.
    *   `HOWTO_GITHUB.md`: **[가이드]** 깃허브 연동 및 사용법을 정리한 현재 문서입니다.
    *   `docs/`: 프로젝트 관련 상세 기술 문서들이 담긴 폴더입니다.

## 4. 🧪 테스트 코드 (`tests/`)
코드의 정상 작동 여부를 확인하는 테스트 파일들입니다.
*   **위치**: `tests\` 폴더 전체 (d:\빅프로젝트\tests\)
*   **주요 파일**:
    *   유닛 테스트 및 통합 테스트 코드들이 포함됩니다.

---

## 🚫 5. 중요: 업로드에서 제외된 파일들 (Excluded)
아래 파일들은 **보안** 및 **용량** 문제로 깃허브에 **절대 올라가지 않습니다.** (`.gitignore` 설정에 의해 자동 제외됨)

1.  **비밀 파일**: `.env` (API Key 등 해킹 위험 정보)
2.  **대용량 데이터**: `data/` 폴더 전체, `*.geojson`, `*.tif`, `*.h5` 등 (GitHub 용량 제한 초과)
3.  **가상환경**: `.venv/`, `venv/` (내 컴퓨터 전용 실행 환경)
4.  **시스템 파일**: `__pycache__`, `.DS_Store` 등

**결론:** 위 목록에 있는 **코드와 문서**만 깔끔하게 `git push` 됩니다. 안심하고 진행하셔도 됩니다.
